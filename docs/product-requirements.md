# SwipeClean Product Requirements

## Goal

Help users clean photo and video libraries quickly with a swipe-based review flow.

## Initial Platform

Android first. iOS support is planned after Android media access, deletion review, and core retention metrics are validated.

## MVP

- Show a queue of photos and videos.
- Swipe or tap to keep or mark for deletion.
- Track review progress.
- Confirm before destructive deletion.
- Save cleanup session data for authenticated users.

## Non-Goals

- Cloud backup of original media in the MVP.
- Automated AI deletion without user review.
- iOS-specific media handling in the first Android release.
